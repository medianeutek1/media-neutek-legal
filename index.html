TikTok Integration

Direct publishing to TikTok

ShortsAI integrates with TikTok via the official Content Posting API and Login Kit. Users connect their TikTok account through TikTok's OAuth flow, and videos are published directly to their profile.

  • Users log in via TikTok Login Kit — no passwords stored
  • Videos posted directly to the user's TikTok profile using video.publish
  • Draft uploads supported via video.upload for creator review before posting
  • User profile info fetched via user.info.basic for account display only
  • All tokens stored securely and never shared with third parties

Permissions Used

user.info.basic video.publish video.upload
1. User clicks "Connect TikTok"
2. Redirected to TikTok OAuth consent screen
3. User approves permissions
4. ShortsAI receives access token
5. Video uploaded via Content Posting API
6. Post published to user's TikTok profile
